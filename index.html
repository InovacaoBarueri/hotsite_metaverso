  • O que é o
    metaverso?

    O metaverso é um novo universo totalmente online, onde proporciona uma experiência imersiva com o uso das tecnologias de realidade virtual. Dentro dele é possível interagir com o avatar de outras pessoas, jogar e até mesmo trabalhar.

    Acesse sem pagar nada

    Você pode entar no metaverso de maneira gratuita com a plataforma Mozilla Hubs.

    Precisa do óculos para entrar?

    Para entrar no metaverso não é obrigatório o uso do óculos de realidade aumentada. Mas para uma melhor experiência o uso dele é indispensável.
